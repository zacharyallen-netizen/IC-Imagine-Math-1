📝 Unit 7 Guided Notes — Quadratic Functions

Every parabola has a vertex and an axis of symmetry. Read the form, find the features.

Standards: F-IF.4/7/9 · A-CED.1/2 · F-BF.1b

① Features ② Graphing ③ Domain/Range ④ Create ⑤ Compare ⑥ Build

1 Key Features of a Parabola

📌 Vertex = max or min. Axis of symmetry = vertical line through the vertex. x-intercepts = zeros. Opens up (a > 0, has a min) or down (a < 0, has a max).
y = (x−1)² − 4 : vertex (1,−4), axis x = 1, zeros −1 & 3, y-int −3

2 Graphing Quadratics

📌 From factored form: plot the zeros, find the axis = midpoint of zeros, then the vertex, then a couple of symmetric points.

Graph y = x² − 2x − 3

Factor → zeros: (x − 3)(x + 1) → click to reveal x = 3, x = −1
Axis = midpoint: (3 + (−1))/2 = click to reveal x = 1
✓ Vertex (plug x = 1): click to reveal (1, −4)

3 Domain, Range & End Behavior

📌 Domain = all real numbers. Range = y ≥ min (opens up) or y ≤ max (opens down). End behavior: both ends up (a > 0) or both down (a < 0).
y = −x² + 4 opens down: max (0,4), range y ≤ 4

4 Creating Quadratic Equations

📌 Area and projectile problems are quadratic. Set up, then factor or take roots.
MODEL A rectangle’s length is 2 more than its width and the area is 24. Find the width.
Let w = width → w(w + 2) = 24 → w² + 2w − 24 = 0 → (w + 6)(w − 4) = 0 → w = 4 (length 6).

5 Comparing Functions

📌 Compare features (max/min, intercepts) even across forms — a graph, a table, and an equation. Find the same feature for each, then compare.

Function A (graph) has a max of 5. Function B: y = −x² + 3. Which has the greater max?

B’s vertex is (0, 3) → max = 3. Compare with A’s max 5 → click to reveal A is greater (5 > 3)

6 Building Functions

📌 Build a new function by adding or multiplying functions. Multiplying two linear functions gives a quadratic.

l(x) = x + 3 and w(x) = x. Build the area A(x) = l·w.

Multiply: x(x + 3) = click to reveal x² + 3x
