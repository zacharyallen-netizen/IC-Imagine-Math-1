1 What Is a Function?

📌 A function gives exactly one output for each input. (No input may map to two different outputs.)
Input x goes in, the rule f produces one output f(x)

Vertical line test: a graph is a function if no vertical line crosses it more than once.

2 Function Notation

📌 f(x) is the output for input x. f(3) means “evaluate at x = 3.” The statement f(3) = 7 reads “the input 3 gives output 7.”

5 Domain & Range

📌 Domain = all inputs (x-values). Range = all outputs (y-values). Discrete = separate points; continuous = unbroken line.
In context, pick a reasonable domain — for example, time can’t be negative, and people must be whole numbers.

6 Average Rate of Change & Parameters

📌 Average rate of change = Δy/Δx = (f(b) − f(a))/(b − a). For a line, that’s the slope. In context: slope = rate, intercept = starting value.
