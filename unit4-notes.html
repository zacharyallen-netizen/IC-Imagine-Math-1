1 Sequences Are Functions

📌 A sequence is a function whose inputs are the term numbers 1, 2, 3, … Add → arithmetic → linear. Multiply → geometric → exponential.

2 Arithmetic Sequences

📌 Add the common difference d. Explicit: aₙ = a₁ + (n − 1)d. Recursive: aₙ = aₙ₋₁ + d.

3 Geometric Sequences

📌 Multiply by the common ratio r. Explicit: aₙ = a₁ · r⁽ⁿ⁻¹⁾. Recursive: aₙ = aₙ₋₁ · r.

Sequence 2, 6, 18, 54, … Find a₅.

r = 3, a₁ = 2 → explicit: aₙ = click to reveal 2 · 3⁽ⁿ⁻¹⁾
✓ a₅ = 2 · 3⁴ = 2 · 81 = click to reveal 162

4 Linear vs. Exponential

📌 Constant difference → linear. Constant ratio → exponential. Exponential growth starts slow, then rockets up.
y = 2x + 1 (line) vs. y = 2ˣ (curve) — the curve overtakes the line

5 Growth & Decay

📌 y = a · bˣ. a = initial value. b = growth factor. b > 1 → growth; 0 < b < 1 → decay. Rate: growth b = 1 + r, decay b = 1 − r.
Growth: y = 1.5ˣ
Decay: y = 10·(0.6)ˣ

$500 grows 4% per year. Write the model.

a = 500, r = 0.04 → b = 1 + 0.04: click to reveal b = 1.04
✓ Model: click to reveal y = 500(1.04)ˣ

6 Key Features & End Behavior

📌 Exponentials have a horizontal asymptote (usually y = 0), domain all real numbers, and range y > 0 (when a > 0). Exponential growth eventually exceeds any linear or quadratic.
